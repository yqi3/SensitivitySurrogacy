# SensitivitySurrogacy (Python)

Python implementation of the SensitivitySurrogacy R package for sensitivity analysis and partial identification of long-term treatment effects using surrogate outcomes.

## Installation

```bash
git clone https://github.com/yqi3/SensitivitySurrogacy.git
cd SensitivitySurrogacy/python
pip install --upgrade pip
pip install .
```

It is recommended to install in a virtual environment:
```bash
python3 -m venv .venv
source .venv/bin/activate    # On Windows: .venv\Scripts\activate
pip install --upgrade pip
pip install .
